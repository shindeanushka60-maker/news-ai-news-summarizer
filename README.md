# 📰 AI News Summarizer

An automated n8n workflow that fetches latest news from RSS feeds, summarizes them using AI, and sends a daily email digest.

## ⚙️ Features
- RSS news fetching  
- AI-powered summarization  
- Email delivery  
- Fully automated workflow  

## 🚀 Setup
1. Import `myworkflow.json` into n8n  
2. Add API keys (AI + Gmail)  
3. Set RSS feed URL  
4. Run workflow
